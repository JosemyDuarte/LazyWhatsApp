# WhatsAI Development Guidelines

## Active Technologies

- TypeScript 5.x, Astro 4.x + Astro, Vitest, Playwright, Web Workers API, LLM SDKs (Ollama, Cloudflare AI, etc.)

## Commands

npm test: Run tests
npm run lint: Run linter
npm run build: Build the app
npm run dev: Run the app in development mode

## Code Style

TypeScript 5.x, Astro 4.x: Follow standard conventions
